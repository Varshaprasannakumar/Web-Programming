HTML is the standard markup language for creating Web pages, it describes the structure of a Web page. HTML is the language in which most websites are written, it is used to create pages and make them functional. Hypertext means that the document contains links that allow the reader to jump to other places in the document or to another document altogether. A Markup Language is a way that computers speak to each other to control how text is processed and presented.Tags and attributes are the basis of HTML.
